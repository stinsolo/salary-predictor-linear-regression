# Salary Predictor using Simple Linear Regression

## 📌 Project Overview
This project predicts the **salary of a software engineer** based on their **years of experience** using **Simple Linear Regression**, a basic machine learning algorithm.

The model is trained on a **real-world dataset from Kaggle** and demonstrates the complete machine learning workflow from data loading to prediction.

---

## 🎯 Objective
- To understand the basics of machine learning
- To implement Simple Linear Regression
- To predict salary using years of experience as input
- To work with a real-world dataset

---

## 📊 Dataset
- Source: Kaggle
- Type: CSV file
- Features used:
  - `YearsExperience` – Independent variable (input)
  - `Salary` – Dependent variable (output)

---

## 🛠 Tools & Technologies
- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## ⚙️ Methodology
1. Import required Python libraries
2. Load the dataset using Pandas
3. Visualize the data to understand the relationship
4. Split the dataset into training and testing sets
5. Train a Simple Linear Regression model
6. Evaluate the model performance
7. Predict salary for new experience values

---

## 📈 Model Used
**Simple Linear Regression**

Mathematical equation:
Salary = m × Experience + c
Where:
- `m` is the slope
- `c` is the intercept

---

## ✅ Results
- The model successfully learned the relationship between experience and salary
- It can predict salary for unseen experience values
- Example:
  - For **7 years of experience**, the model predicted a salary of approximately **90,000** (dataset dependent)

---

## 🌍 Applications

## ⚠️ Limitations
- Only one feature (experience) is considered
- Real salary depends on multiple factors such as skills, location, and company
- Assumes a linear relationship between experience and salary
